    [NOIP1998 提高组] 车站¶

    约 464 个字 48 行代码 预计阅读时间 2 分钟

    题目描述¶

    火车从始发站（称为第 \(1\) 站）开出，在始发站上车的人数为 \(a\)，然后到达第 \(2\) 站，在第 \(2\) 站有人上、下车，但上、下车的人数相同，因此在第 \(2\) 站开出时（即在到达第 \(3\) 站之前）车上的人数保持为 \(a\) 人。从第 \(3\) 站起（包括第 \(3\) 站）上、下车的人数有一定规律：上车的人数都是前两站上车人数之和，而下车人数等于上一站上车人数，一直到终点站的前一站（第 \(n-1\) 站），都满足此规律。现给出的条件是：共有 \(n\) 个车站，始发站上车的人数为 \(a\)，最后一站下车的人数是 \(m\)（全部下车）。试问 \(x\) 站开出时车上的人数是多少？

    输入格式¶

    输入只有一行四个整数，分别表示始发站上车人数 \(a\)，车站数 \(n\)，终点站下车人数 \(m\) 和所求的站点编号 \(x\)。

    输出格式¶

    输出一行一个整数表示答案：从 \(x\) 站开出时车上的人数。

    样例 #1¶

    样例输入 #1¶

    5 7 32 4

    样例输出 #1¶

    13

    提示¶

    对于全部的测试点，保证 \(1 \leq a \leq 20\)，\(1 \leq x \leq n \leq 20\)，\(1 \leq m \leq 2 \times 10^4\)。

    NOIP1998 提高组 第一题

    Solution¶

    • 我不会数学，考虑暴力枚举
    • 变量主要是第二站上下车人数，考虑枚举它
      • 由题意\(m<=20000\),枚举到20000即可
      • 题目坑人

      • 在第 2 站有人上、下车

      • 其实可能为0人

    之后模拟即可

    20*20000=400000次模拟，时间复杂度为\(O(nm)\)

    变量解释¶

    • pp (pre pre) 前前站上车数
    • p (pre) 前站上车数
    • up 当前站上车数
    • dw 当前站下车数
    • cur 当前站出发后车上人数
    • ans 所求站点车上人数

    Source code¶

    #include <stdio.h>
#include <string.h>
#include <math.h>
#define ll long long
int main()
{
    int a,n,m,x,i,j;

    scanf("%d%d%d%d",&a,&n,&m,&x);
    ll pp=a,p=a,up,dw,cur=a,ans;

    if(x<=2)
    {
        printf("%d\n",a);
        return 0;
    }
    if(x==n)
    {
        printf("0\n");
        return 0;
    }
    int fl=0;
    for(ll dw2=0;dw2<=20000;dw2++)//枚举第二站下车人数
    {
        pp=a,p=dw2,cur=a;//注意每次初始化
        for(i=3;i<n;i++)
        {
            dw=p;
            up=pp+p;
            pp=p;   //记得更新前前站     
            p=up;
            cur+=up;
            cur-=dw;
            if(i==x)
            {
                ans=cur;
            }
        }
        if(cur==m)
        {
            fl=1;
            break;
        }
    }
    printf("%lld\n",ans);
    return 0;

}
